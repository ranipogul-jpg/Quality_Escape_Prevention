# Quality Escape Prevention

## Project Overview

This project predicts whether a manufactured product requires inspection before dispatch using Machine Learning.

## Problem Statement

Quality defects are sometimes detected too late. This model helps identify products that should be inspected before shipment.

## Machine Learning Algorithm

Random Forest Classifier

## Features

- Air Temperature
- Process Temperature
- Rotational Speed
- Torque
- Tool Wear
- Machine Type

## Output

Inspection Required

or

No Inspection Required

## Technologies Used

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Joblib
